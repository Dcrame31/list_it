# List It!

Simple app for creating lists for whatever your heart desires. Lists are organized in categories, which may be created, edited, and deleted. 

## Getting Started

To use this app:

* clone file
* run bundle
* rake db:migrate
* run shotgun
* copy and paste given url in browser e.g. 'localhost:9292'
